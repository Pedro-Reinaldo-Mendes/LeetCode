# 121. Best Time to Buy and Sell Stock

[LeetCode Problem Link](https://leetcode.com/problems/best-time-to-buy-and-sell-stock/)

## Performance

- **Runtime**: 1 ms (Beats 99.87%)
- **Memory**: 94.60 MB (Beats 44.23%)

## Complexity Analysis

- **Time Complexity**: $O(n)$, where $n$ is the number of days (length of the `prices` array). We make a single pass through the array.
- **Space Complexity**: $O(1)$, as we only store two integers (`maxProfit` and `minBuyValue`).

## Approach

The approach uses a **one-pass** greedy strategy to find the maximum possible profit.

1.  **Initialize Variables**: Set `maxProfit` to 0 and `minBuyValue` to the maximum possible integer value.
2.  **Iterate through Prices**:
    *   For each day's price:
        *   If the current price is less than `minBuyValue`, update `minBuyValue` to the current price. (We found a cheaper day to buy).
        *   Otherwise, calculate the potential profit if we sold today (`price - minBuyValue`). If this profit is greater than `maxProfit`, update `maxProfit`.
3.  **Result**: Return the `maxProfit` found.

### Solution.java

```java
class Solution {
    public int maxProfit(int[] prices) {
        int maxProfit = 0;
        int minBuyValue = Integer.MAX_VALUE;

        for (int price : prices) {
            if (price < minBuyValue) minBuyValue = price;
            else if (price - minBuyValue > maxProfit) maxProfit = price - minBuyValue;
        }

        return maxProfit;
    }
}
```
