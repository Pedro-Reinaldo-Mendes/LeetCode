# LeetCode - 3484. Design Spreadsheet

**Profile:** [PedroRM](https://leetcode.com/u/PedroRM/)

## Performance

- **Runtime:** 121 ms (Beats 58.93%)
- **Memory Usage:** 84.40 MB (Beats 36.83%)

## Complexity Analysis

- **Time Complexity:**
  - **Constructor:** $O(R \times C)$, where $R$ is the number of rows and $C$ is the number of columns (fixed at 26).
  - **setCell / resetCell:** $O(1)$ amortized, as it involves basic string parsing and array access.
  - **getValue:** $O(L)$, where $L$ is the length of the formula string, as we scan the formula once to split it and then perform lookups.
- **Space Complexity:** $O(R \times C)$ to store the spreadsheet data in a 2D matrix.

## Logic Description

This problem asks us to design a basic spreadsheet that supports setting cell values, resetting them, and evaluating simple formulas (sums of two components).

1.  **Storage**: A 2D integer array `matrix[rows][26]` is used to store the values. Rows are 0-indexed while columns are mapped from 'A'-'Z' to 0-25.
2.  **Initialization**: Cells are initialized to `-1` to represent an empty or uninitialized state.
3.  **Cell Mapping**: A helper function `mapCell` converts a string coordinate like "A1" into row/column indices.
4.  **Formula Evaluation**:
    - The `getValue` method parses a formula (e.g., "=A1+10" or "=5+B2").
    - It identifies the `+` sign to split the formula into two components.
    - Each component is either a cell reference (starting with an uppercase letter) or a literal integer.
    - It retrieves the numeric value for each part (defaulting to 0 if the cell value is -1) and returns their sum.

### Solution.java

```java
import java.util.Arrays;

/**
 * 3484. Design Spreadsheet
 *
 * @see <a href="https://leetcode.com/problems/design-spreadsheet/">Design Spreadsheet</a>
 */
class Spreadsheet {

    private int[][] matrix;

    public Spreadsheet(int rows) {
        matrix = new int[rows][26];
        for (int[] col : matrix) {
            Arrays.fill(col, -1);
        }
    }
    
    public void setCell(String cell, int value) {
        int[] cellPos = mapCell(cell);
        matrix[cellPos[0]][cellPos[1]] = value;
    }
    
    public void resetCell(String cell) {
        setCell(cell, 0);
    }
    
    public int getValue(String formula) {
        // Search for the '+' symbol
        int plusSymI = 1;
        while (plusSymI < formula.length()) {
            if (formula.charAt(++plusSymI) == '+') break;
        }

        // Define both components to be summed (as strings)
        String compX = formula.substring(1, plusSymI);
        String compY = formula.substring(plusSymI + 1, formula.length());

        // Process each components numeric values
        int compXNum = getNumericValue(compX);
        if (compXNum == -1) compXNum = 0;
        int compYNum = getNumericValue(compY);
        if (compYNum == -1) compYNum = 0;
        
        // Return the sum
        return compXNum + compYNum;
    }

    private static int[] mapCell(String cell) {
        int[] res = new int[2];

        // Set the column
        res[1] = cell.charAt(0) - 'A';

        // Set the row
        res[0] = Integer.parseInt(cell.substring(1, cell.length())) - 1;

        return res;
    }

    private int getNumericValue(String str) {
        if (Character.isUpperCase(str.charAt(0))) {
            int[] cellPos = mapCell(str);
            return matrix[cellPos[0]][cellPos[1]];
        } else {
            return Integer.parseInt(str);
        }
    }
}
```
