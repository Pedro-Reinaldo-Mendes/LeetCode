# 412. Fizz Buzz

[LeetCode Problem Link](https://leetcode.com/problems/fizz-buzz/)

## Performance

- **Runtime**: 1 ms (Beats 99.94%)
- **Memory**: 46.64 MB (Beats 90.12%)

## Complexity Analysis

- **Time Complexity**: $O(n)$, where $n$ is the input integer. We iterate from $1$ to $n$ exactly once.
- **Space Complexity**: $O(n)$, to store the resulting list of strings.

## Approach

The approach uses a straightforward loop to check divisibility conditions in the correct order.

1.  **Iterate from 1 to n**: Use a `for` loop to process each number from $1$ up to $n$ (inclusive).
2.  **Check Divisibility**:
    *   **Divisible by both 3 and 5 (15)**: Append "FizzBuzz".
    *   **Divisible by 3 only**: Append "Fizz".
    *   **Divisible by 5 only**: Append "Buzz".
    *   **Otherwise**: Append the string representation of the number itself.
3.  **Result**: Return the list of strings.

### Solution.java

```java
class Solution {
    public List<String> fizzBuzz(int n) {
        List<String> res = new ArrayList<>();
        for (int i = 1; i <= n; i++) {
            if (i % 15 == 0) res.add("FizzBuzz");
            else if (i % 3 == 0) res.add ("Fizz");
            else if (i % 5 == 0) res.add("Buzz");
            else res.add(String.valueOf(i));
        }
        return res;
    }
}
```
