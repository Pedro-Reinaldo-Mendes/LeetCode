# 543. Diameter of Binary Tree

[LeetCode Problem Link](https://leetcode.com/problems/diameter-of-binary-tree/)

## Performance

- **Runtime**: 0 ms (Beats 100.00%)
- **Memory**: 42.06 MB (Beats 87.23%)

## Complexity Analysis

- **Time Complexity**: $O(N)$, where $N$ is the number of nodes in the tree. We visit each node exactly once during the DFS traversal.
- **Space Complexity**: $O(H)$, where $H$ is the height of the tree, due to the recursion stack. In the worst case (skewed tree) this is $O(N)$, and in the best case (balanced tree) this is $O(\log N)$.

## Approach

The diameter of a binary tree is the length of the longest path between any two nodes. This path may or may not pass through the root.

1.  **DFS (Post-order traversal)**: For each node, we compute the height of its left and right subtrees recursively.
2.  **Diameter at each node**: The diameter passing through a given node equals the total number of edges in the longest path between any two leaves that goes through that node. Since `dfs` returns the number of edges from a node to its deepest descendant (using `-1` for `null` as a sentinel), `leftHeight + 1` gives the edges from the current node to its deepest left leaf (adding the edge to the left child), and similarly `rightHeight + 1` for the right side. The full diameter is therefore `(leftHeight + 1) + (rightHeight + 1) = leftHeight + rightHeight + 2`.
3.  **Track global maximum**: We keep a class-level variable `maxDiameter` and update it whenever we find a larger diameter at any node.
4.  **Height returned**: Each recursive call returns `1 + max(leftHeight, rightHeight)`, which is the height of the current subtree, allowing parent nodes to compute their own diameter.


### Solution.java

```java
/**
 * 543. Diameter of Binary Tree
 *
 * https://leetcode.com/problems/diameter-of-binary-tree/
 */
class Solution {
    int maxDiameter = 0;

    public int diameterOfBinaryTree(TreeNode root) {
        dfs(root);
        return maxDiameter;
    }

    private int dfs(TreeNode node) {
        if (node == null) return -1;
        int leftHeight = dfs(node.left);
        int rightHeight = dfs(node.right);
        int diameter = leftHeight + rightHeight + 2;
        if (diameter > maxDiameter) maxDiameter = diameter;
        return 1 + ((leftHeight > rightHeight) ? leftHeight : rightHeight);
    }
}
```
