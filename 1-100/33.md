# 33. Search in Rotated Sorted Array

[LeetCode Problem Link](https://leetcode.com/problems/search-in-rotated-sorted-array)

## Performance

- **Runtime**: 0 ms (Beats 100.00%)
- **Memory**: 44.21 MB (Beats 8.09%)

## Complexity Analysis

- **Time Complexity**: $O(\log N)$, where $N$ is the length of the array. The algorithm uses a modified binary search.
- **Space Complexity**: $O(1)$, as we only use a constant amount of extra space.

## Approach

The solution uses a modified binary search to locate the target in the rotated sorted array.

1.  **Initial Check**: We compare the target with the first element (`nums[0]`). If it matches, we return 0. We also determine if the target is "high" (greater than `nums[0]`) or "low".
2.  **Binary Search**: We initialize pointers `l` and `r` and enter a loop.
3.  **Midpoint Logic**: At each step, we calculate the midpoint `i`. If `nums[i]` matches the target, we return the index.
4.  **Shift Logic**: We adjust `l` or `r` based on the values of the target, `nums[i]`, and the first element `x` to correctly navigate the rotated segments.
    - If targeting a value > start and `target < nums[i]`, we go left.
    - If targeting a value < start and `target > nums[i]`, we go right.
    - Otherwise, we use the value of `nums[i]` relative to the start to decide direction.

### Solution.java

```java
/**
 * 33. Search in Rotated Sorted Array
 * 
 * https://leetcode.com/problems/search-in-rotated-sorted-array
 */
class Solution {
    public int search(int[] nums, int target) {
        int x = nums[0];
        if (x == target) return 0;

        boolean highTarget = target > x;

        int l = 1;
        int r = nums.length;

        while (l != r) {
            int i = ((r - l) / 2) + l;
            if (target == nums[i]) return i;
            if (highTarget && target < nums[i]) r = i;
            else if (!highTarget && target > nums[i]) l = i + 1;
            else if (nums[i] > x) l = i + 1;
            else r = i;
        }

        return -1;
    }
}
```
