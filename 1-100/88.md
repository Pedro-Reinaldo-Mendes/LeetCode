# 88. Merge Sorted Array

[LeetCode Problem Link](https://leetcode.com/problems/merge-sorted-array/)

## Performance

- **Runtime**: 0 ms (Beats 100.00%)
- **Memory**: 43.45 MB (Beats 96.72%)

## Complexity Analysis

- **Time Complexity**: $O(m + n)$, where $m$ and $n$ are the lengths of the two arrays. We iterate through both arrays once in the worst case.
- **Space Complexity**: $O(1)$. The merge is performed in-place without using any additional data structures.

## Approach

The problem requires merging two sorted arrays where `nums1` has enough space to hold both arrays. The key insight is to fill `nums1` from right to left, starting from the end.

1. **Two Pointers Strategy**: We use two pointers `i` and `k` to track positions in `nums1` (actual elements) and `nums2` respectively.
   - `i` starts at `m - 1` (last actual element in `nums1`)
   - `k` starts at `n - 1` (last element in `nums2`)

2. **Reverse Iteration**: We iterate through `nums1` from right to left (index `j` from `nums1.length - 1` to `0`), filling positions with the larger element between `nums1[i]` and `nums2[k]`.

3. **Handle Edge Cases**:
   - If `i < 0`: All elements from `nums1` are placed, so copy remaining elements from `nums2`.
   - If `k < 0`: All elements from `nums2` are placed, remaining elements in `nums1` are already in correct position, so break.
   - Otherwise, compare `nums1[i]` and `nums2[k]` and place the larger one at position `j`.

4. **Why This Works**: By filling from right to left, we avoid overwriting elements in `nums1` that haven't been processed yet.

### Solution.java

```java
/**
 * 88. Merge Sorted Array
 *
 * https://leetcode.com/problems/merge-sorted-array/
 */
class Solution {
    public void merge(int[] nums1, int m, int[] nums2, int n) {

        // [1,8,9,0,0] ; m = 3
        //      /\  /\

        // [2,5] ; n = 2
        //    /\

        int i = m - 1;
        int k = n - 1;

        for (int j = nums1.length - 1; j >= 0; j--) {

            if (i < 0) {
                for (;j>=0;j--) nums1[j] = nums2[k--];
            } else if (k < 0) {
                break;
            } else if (nums1[i] > nums2[k]) {
                nums1[j] = nums1[i--];
            } else {
                nums1[j] = nums2[k--];
            }
        }

    }
}
```
