# 824. Goat Latin

[LeetCode Problem Link](https://leetcode.com/problems/goat-latin/)

## Performance

- **Runtime**: 2 ms (Beats 89.89%)
- **Memory**: 43.08 MB (Beats 91.16%)

## Complexity Analysis

- **Time Complexity**: $O(N + W^2)$, where $N$ is the length of the input sentence and $W$ is the number of words. Each word is processed once, and for each word $i$, we append $i$ additional 'a' characters, leading to a quadratic sum.
- **Space Complexity**: $O(N + W^2)$, to store the resulting sentence with the appended characters in a `StringBuilder`.

## Approach

The approach involves parsing the sentence into words and applying the Goat Latin rules to each word individually.

1.  **Split the Sentence**: Break the sentence into individual words using the space character as a delimiter.
2.  **Define Vowels**: Maintain a set or array of vowels (including both cases) for quick lookup.
3.  **Process Each Word**:
    *   Check if the first character of the word is a vowel.
    *   **Rule 1**: If it begins with a vowel, the word remains the same.
    *   **Rule 2**: If it begins with a consonant, move the first character to the end of the word.
    *   **Rule 3**: Append "ma" to the end of the (possibly modified) word.
    *   **Rule 4**: Append 'a' characters based on the word's position in the sentence (e.g., one 'a' for the first word, two for the second, etc.).
4.  **Join and Return**: Combine the processed words with spaces and return the final string.

### Solution.java

```java
class Solution {
    public String toGoatLatin(String sentence) {
        String[] words = sentence.split(" ");
        char[] vowels = {'a','e','i','o','u','A','E','I','O','U'};
        StringBuilder sb = new StringBuilder();

        int x = 1;
        for (String word : words) {
            boolean wordHasVowel = false;
            for (char vowel : vowels) {
                if (word.charAt(0) == vowel) {
                    wordHasVowel = true;
                    break;
                }
            }

            if (!wordHasVowel) {
                for (int i = 1; i <= word.length(); i++) {
                    if (word.length() == i) sb.append(word.charAt(0));
                    else sb.append(word.charAt(i));
                }
            } else sb.append(word);

            sb.append("ma");

            for (int a = 0; a < x; a++) {
                sb.append("a");
            }
            x++;

            if (word != words[words.length - 1]) sb.append(" ");
        }

        return sb.toString();
    }
}
```
