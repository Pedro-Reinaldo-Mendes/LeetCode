# 2109. Adding Spaces to a String

[LeetCode Problem Link](https://leetcode.com/problems/adding-spaces-to-a-string/)

## Performance

- **Runtime**: 21 ms (Beats 77.46%)
- **Memory**: 90.18 MB (Beats 68.39%)

## Complexity Analysis

- **Time Complexity**: $O(N + M)$, where $N$ is the length of the string `s` and $M$ is the number of spaces to add. We iterate through the string once and insert spaces as needed.
- **Space Complexity**: $O(N + M)$, to store the resulting string in a `StringBuilder`.

## Approach

The approach uses a **Two-Pointer/Stream** style to construct the new string efficiently.

1.  **StringBuilder**: Initialize a `StringBuilder` to hold the final string.
2.  **Pointer for spaces**: Use `spacesI` to track the current index in the `spaces` array.
3.  **Construct string**:
    *   Iterate through each character of the original string `s`.
    *   Before appending the character, check if its index `i` matches the next value in `spaces[spacesI]`.
    *   If it matches, append a space and increment `spacesI`.
    *   Append the character from `s`.
4.  **Result**: Convert the `StringBuilder` to a string and return it.

### Solution.java

```java
class Solution {
    public String addSpaces(String s, int[] spaces) {
        
        StringBuilder sb = new StringBuilder();
        int spacesI = 0;

        for (int i = 0; i < s.length(); i++) {
            if (spacesI < spaces.length && spaces[spacesI] == i) {
                sb.append(' ');
                spacesI++;
            }
            sb.append(s.charAt(i));
        }

        return sb.toString();
    }
}
```
